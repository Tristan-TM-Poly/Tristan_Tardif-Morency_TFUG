# META ORCHESTRATOR COMMAND POST

This folder turns the existing TRISTAN2 / AI-7 runtime-bus doctrine into a governed meta-orchestration surface for:
- ZIP intake
- GitHub repositories
- Google Drive docs/sheets/boards
- interactive dashboard site
- cloud/server jobs
- external web knowledge intake

## Core law
Do not create a second disconnected command system.
Use the existing Orchestrator Console as the sovereign command surface and treat this folder as the versioned implementation layer.

## System split
- TRISTAN2 Core = governance, scoring, canon policy, promotion, rollback, quarantine
- AI-7 Runtime = execution, synchronization, routing, observation, propagation
- Meta Orchestrator Command Post = cockpit over adapters, domain packs, campaigns, waves, and return routes
