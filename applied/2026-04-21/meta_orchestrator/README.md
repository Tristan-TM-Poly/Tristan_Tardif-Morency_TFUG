# TRISTAN2 Meta-Orchestrator Anchor

This directory anchors a governed meta-orchestrator control surface for TFUGA / AI7 / TRISTAN2.

## Intent
Create one reversible bridge between:
- local ZIP pipelines
- Drive command boards and manuals
- GitHub repo workflows and artifacts
- future web/cloud adapters

## Confirmed control chain
User -> ChatGPT Adapter -> TRISTAN2 Core -> AI7 Runtime Bus -> Local Runtime -> Drive / GitHub / Web / Cloud

## Hard rules
1. No promotion without score, review, and rollback pointer.
2. Fractal-Loop remains the first bounded pilot.
3. Every command must map to one primary lane.
4. Prefer small auditable mutations.

- `system.yaml`: control-plane registry
- `README.md`: human-facing anchor
