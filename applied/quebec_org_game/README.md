# Quebec Organization Exploration Game

This module defines a coupled data system and gameplay layer to explore organizations in Quebec through evidence-based analysis.

## Philosophy
- Facts are separated from claims
- Every statement must be backed by sources
- Scoring is multidimensional and reversible
- Uncertainty is explicit and tracked

## Core Layers
1. Data layer (organizations, events, sources)
2. Evidence layer (claims, source links)
3. Scoring layer (dimension-based evaluation)
4. Governance layer (flags, moderation)
5. Gameplay layer (quests, profiles)

## Gameplay Concept
Players act as investigators:
- Discover organizations
- Link events and sources
- Evaluate impacts across dimensions
- Resolve conflicting evidence
- Build trustworthy profiles

## Integration
Designed to connect with:
- TFUGA HGFM graphs
- AI-7 scoring systems
- Drive-based knowledge layers

## Status
Initial schema deployed. Ready for data ingestion and gameplay engine integration.
