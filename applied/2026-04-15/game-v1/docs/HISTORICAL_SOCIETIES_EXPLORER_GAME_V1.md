# Historical Societies Explorer Game v1

## Purpose
This game module lets players explore historical societies through evidence, context, consequences, and moral complexity.

## Core law
The game does not label an entire society as good or bad.
It examines actions, institutions, innovations, harms, tradeoffs, and consequences.

## Learning goals
- understand that societies contain both constructive and destructive actions
- separate evidence from legend or simplification
- compare short-term gains with long-term consequences
- trace how institutions shape human flourishing or harm
- practice disciplined historical judgment

## Main loop
Trace -> Context -> Action -> Consequence -> Comparison -> Review -> Public Summary

## Core objects
- society profiles
- action cards
- consequence maps
- evidence cards
- review gates
- comparison missions

## Player tasks
- inspect a society profile
- open evidence-backed action cards
- classify actions by domain and consequence
- compare beneficial and harmful outcomes
- detect oversimplified judgments
- pass a review gate before publishing a conclusion

## Domain clusters
- governance
- law
- war
- knowledge
- infrastructure
- trade
- social inclusion
- slavery and domination
- ecology and resource use
- ritual and worldview

## Example societies for v1
- Ancient Egypt
- Classical Athens
- Roman Republic and Empire
- Han China
- Maya civilization
- Medieval Mali
- Abbasid Caliphate
- Haudenosaunee Confederacy

## Review doctrine
Every public-facing conclusion must distinguish:
- established evidence
- plausible interpretation
- contested judgment
- simulated extension

## Final law
The Historical Societies Explorer should teach that historical understanding grows stronger when evidence, context, and moral clarity are kept together without collapsing complexity.
