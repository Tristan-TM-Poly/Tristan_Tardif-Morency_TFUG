# Lost Technologies Explorer Game v1

## Purpose
This module lets players reconstruct, test, compare, review, and teach candidate lost technologies without collapsing evidence, hypothesis, and simulation into one category.

## Core law
A lost technology candidate is not an established fact until it survives evidence review, bounded modeling, and anti-inflation checks.

## Learning goals
- distinguish evidence-backed technical traces from speculative extrapolation
- compare multiple reconstruction paths for the same candidate technology
- understand tradeoffs between function, symbolism, materials, and governance
- practice disciplined promotion and downgrade decisions
- learn that non-validation is also meaningful knowledge

## Main loop
Trace -> Reconstruction Candidate -> Material/Function Comparison -> Simulation -> Review Gate -> Promote or Downgrade -> Public Summary

## Core objects
- technology candidates
- evidence fragments
- reconstruction paths
- simulation cards
- review gates
- public summary cards

## Example candidate families for v1
- water management chambers

- resonance-sensitive chamber design
- mechanical transfer and lifting variants
- symbolic-procedural transmission devices

## Review doctrine
Every public-facing conclusion must distinguish:
- evidence-backed traces
- plausible reconstruction
- contested interpretation
- simulated extension

## Final law
The Lost Technologies Explorer teaches how to reconstruct responsibly, how to reject seductive overclaims, and how to turn uncertainty into governed inquiry.
