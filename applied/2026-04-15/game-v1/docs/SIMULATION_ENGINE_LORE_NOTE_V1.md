# Simulation Engine Lore Note v1

## Purpose
This simulation layer is intended to generate bounded scenario branches, civilization arcs, and lore candidates for Alexandrie Fractale.

## Important boundary
These simulations do not establish real human history. They generate playable, reviewable, and governable historical-lore branches for the game world.

## Core use
- generate thousands of scenario branches
- rank them by survival, reactivation, collapse, and promoted knowledge
- select top branches for missions, campaigns, and world events
- feed branch cards into Yggdrasil and the public interactive explorer

## Core law
Scenario branch = traces + reconstruction + review - inflation

## Selection
Prefer branches with:
- high promoted knowledge
- strong reconstruction power
- acceptable collapse pressure
- strong gate scores

## Output families
- age-of-collapse
- age-of-reactivation
- age-of-symbolic-unity
- age-of-machines
- fractured-transition

## Governance
All generated branches remain lore candidates until reviewed and promoted through TRISTAN2.
