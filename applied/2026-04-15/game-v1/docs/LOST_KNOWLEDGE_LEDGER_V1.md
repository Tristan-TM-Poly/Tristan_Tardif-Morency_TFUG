# Lost Knowledge Ledger v1

## Purpose
This ledger tracks structured hypotheses, survivances, recodings, symbolic compression, and candidate proto-technologies without confusing them with established history.

## Rule
This ledger is for governed exploration, not for automatic promotion to fact.

## Required fields per item
- id
- title
- domain_cluster
- object_family
- hypothesis_statement
- supporting_fragments
- comparison_domains
- simulation_readiness
- review_gate_status
- tristan2_status
- speculative_cost
- anti_inflation_notes
- next_test

## Status scale
- trunk
- proto-trunk
- exploratory
- quarantine
- lore-only

## Domain clusters
- myth
- ritual
- temple
- symbol
- text
- proto-technology
- transmission
- survivance

### LK-001
Title: Temples as multi-functional geometric-ritual devices

## Output use
This ledger feeds:
- Hypotheses in Review on the site
- mission seeds in the game
- simulation branches
- anti-inflation gates

## Final law
The Lost Knowledge Ledger is where the system explores what may be real, while still remembering that curiosity is not evidence.
