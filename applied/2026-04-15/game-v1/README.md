# game-v1

## Purpose
This directory is the runtime seed of Alexandrie Fractale: a playable civilization of governed reconstruction of knowledge.

## Core layers
- `data/`: registries and expansion packs
- `lib/`: engines, scoring, review, simulation, orchestration, integration
- `web/`: dashboards, panels, shell, and interactive surfaces
- `docs/`: canon notes, simulation notes, game bible

## Current highlights
- world graph, missions, gates, progression
- NPCs, AI-BOTS, Big Figures, Great Sages
- generator bots and builder bots
- simulation engine, lore composer, campaign planner
- pyramid engine, Rosetta engine, Great Review engine
- GameShell, PyramidDashboard, GreatReviewPanel, SimulationDashboard, LorePanel, CampaignMap

## Integration direction
1. load base bundle
2. integrate expansion pack
3. create runtime
4. push messages from agents
5. run review and simulation layers
6. render shell and dashboards
