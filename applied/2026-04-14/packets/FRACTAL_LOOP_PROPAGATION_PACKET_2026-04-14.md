# FRACTAL_LOOP_PROPAGATION_PACKET — Applied Snapshot (2026-04-14)

## Packet class
- PropagationPacket
- delta packet

## Purpose
Define how the first Fractal-Loop pilot changes human, machine, AI, review, and canon layers without letting the pilot inflate beyond bounded status.

## Delta targets
- Delta_H: concise human explanation
- Delta_M: machine-readable artifact links
- Delta_A: AI-7 taskable summary
- Delta_R: review-facing change note
- Delta_C: canon-facing bounded impact note

## Anti-inflation rule
Propagation is valid only when clarity increases faster than canon cost.

## Required outputs
1. one human summary
2. one machine artifact pointer list
3. one AI taskable summary
4. one review summary line
5. one canon impact line

## Gate
A pilot remains local unless a propagation packet exists.
