# TFUGA_PUBLIC_USER_PROGRESSION_PACKET — Applied Snapshot (2026-04-14)

## Purpose
Define the first bounded user progression packet for the public-interactive TFUGA layer.

## Progression states
- entry: first contact with AT-1 public layer
- explore: bounded mechanic or demo inspection
- compare: compare artifacts, packets, or bounded views
- inspect: open reviewable summaries or public notes
- collaborate: route toward a bounded collaboration action

## Governance fields
- current_state
- visible_module
- next_allowed_actions
- review_gate_placeholder
- conversion_target

## Law
The user progression packet is valid only if public interaction stays bounded, interpretable, and coupled to real governed artifacts and next steps.
