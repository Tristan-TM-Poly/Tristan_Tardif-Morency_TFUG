# FRACTAL_LOOP_REDUCED_MODEL_PACKET — Applied Snapshot (2026-04-14)

## Packet class
- ReducedModelPacket
- bounded pilot packet

## Purpose
Provide the smallest honest executable/analytic kernel for the first Fractal-Loop pilot so that score/review/falsification claims can attach to a concrete reduced object instead of rhetoric.

## Minimal object
- one bounded Fractal-Loop instance
- one reduced state description
- one reduced update law
- one traceable observable
- one termination or boundedness condition

## Required outputs
1. machine-readable reduced object description
2. one re-runnable reduced trace
3. one metric for boundedness
4. one metric for clarity gain
5. one pointer to follow-on Projector packet

## Gate
No Fractal-Loop promotion beyond bounded pilot without an explicit reduced model packet.
