# TFUGA_INTERACTIVE_DEMO_PACKET — Applied Snapshot (2026-04-14)

## Purpose
Define the first bounded packet family for public-interactive TFUGA demos under the TFUGA x AI-7 x TRISTAN2 trinity.

## Required packet members
- HumanPacket: what the demo teaches or reveals
- MachinePacket: bounded demo state and object ids
- AIPacket: AI-7 interaction and adaptation logic
- ReviewPacket: what TRISTAN2 checks before advancing
- PropagationPacket: where the demo routes the user next

## Demo law
A TFUGA interactive demo is valid only if it remains bounded, interpretable, and coupled to real governed artifacts.
