# RUNTIME_TO_LEDGER_EMISSION_SCHEMA — Applied Snapshot (2026-04-14)

## Purpose
Define the smallest governed schema for carrying executable pilot output toward a ledger-facing layer.

## Minimal schema
- artifact_type
- object_name
- source_path
- produced_at_placeholder
- trace_summary
- boundedness_verdict
- benchmark_pointer
- review_pointer
- promotion_pointer

## First target
- Fractal-Loop reduced trace artifact emitted from the staged executable placeholder

## Law
Runtime emission is useful only when it remains small, machine-readable, and governable by the existing review/promotion structure.
