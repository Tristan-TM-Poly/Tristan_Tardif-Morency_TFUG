# LAUNCHER_CLASSIFICATION_TABLE — Applied Snapshot (2026-04-14)

## Sovereign class
- `run_all.py`
  - role: single daily governed execution path
  - routing: trunk-facing
  - packet expectation: full minimum packet family
  - review requirement: yes

## Advanced laboratory class
- `run_max_plus_ultra.py`
  - role: bounded exploratory launch path
  - routing: lab-only
  - packet expectation: bounded exploratory family
  - review requirement: yes before any visibility increase

## Legacy class
- historical ultra launchers
  - role: memory/reference only
  - routing: archive
  - packet expectation: none unless reactivated
  - review requirement: mandatory before re-entry

## Branch-local class
- branch-specific ad hoc launchers
  - role: local tests only
  - routing: hidden
  - packet expectation: branch-local only
  - review requirement: mandatory before promotion

## Classification rule
A launcher is promoted only if it reduces ambiguity, strengthens bounded execution, and improves governed clarity.
