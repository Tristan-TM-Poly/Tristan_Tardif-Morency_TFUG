# PROJECTOR_MAP_EXAMPLE — Applied Snapshot (2026-04-14)

## Purpose
Provide one explicit map-composition example for the Fractal-Loop Projector packet.

## Minimal composition
- object space -> reduced pilot space : `P1`
- reduced pilot space -> observable space : `P2`
- observable space -> review space : `P3`
- composed map: `P = P3 o P2 o P1`

## Visibility law
A projection is acceptable when it increases readable structure without erasing the bounded object it claims to illuminate.

## Failure mode
If the composed map increases descriptive bulk while reducing pilot clarity, the projection fails and remains non-promotable.
