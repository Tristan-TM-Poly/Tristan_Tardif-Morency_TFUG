# Axiom-Tristan Master v10 Safety Gate

This publication index records a local master archive and its source hashes.

Current level: T2, local artifact and local verification record.

Upgrade rules:

- External publication needs an external URL or receipt.
- Deployment needs logs and a rollback path.
- Experimental claims need raw data, calibration, controls, replicates, and uncertainty.
- Legal or institutional steps need human review and a receipt.
- Purchases need explicit budget approval.

This file is a gate: it prevents local artifacts from being mislabeled as external deployment or experimental validation.
