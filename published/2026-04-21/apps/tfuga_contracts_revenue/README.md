# TFUGA Contracts Revenue App

Android + Windows application foundation for contracts, monetary revenue tracking, and governed workflows.

## Scope
- Dashboard KPI
- Contracts
- Revenue events
- Governance rules
- Settings

## Architecture
- Flutter multi-platform foundation
- Intended targets: Android and Windows
- Governance route: packet -> score -> review -> receipt -> decision -> rollback

## Status
This publication folder is the public spine for the app concept and structure. The local ZIP artifact remains the strongest packaged source deliverable in the current workspace.

- Contracts
- Revenue
